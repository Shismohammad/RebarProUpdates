# RebarPro Updates

Civil Engineering Software solutions

## Get started

1. Install dependencies

   ```bash
   npm i
   ```

2. Start the app

   ```bash
   npm run dev
   ```

3. Build the app

   ```bash
   npm run build
   ```

4. Package the app

   ```bash
   npm run dist
   ```
